# odin-playground

Repository where I experiment with the [Odin programming language](https://odin-lang.org/). Expect random stuff — things I'm curious about, small implementations, and general tinkering.

## Projects

- **bmp-viewer** — A simple BMP file viewer built with SDL3. Parses and renders 24-bit and 32-bit BMP files, with automatic vertical flip handling for bottom-up encoded images.

## Purpose

No structure, no roadmap. Just code I find interesting to write.

## Disclaimer

None of the code here is production-ready and may contain bugs. That said, feel free to use any parts of it if you find them useful.
